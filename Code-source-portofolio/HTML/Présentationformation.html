Présentation du BTS SIO

Le BTS-SIO est une formation de type Brevet de Technicien superieur qui porte sur l'informatique, celle-ci se divise en 2 spécialitées, le SLAM et le SISR.

  • Le SLAM est une spécialité de programation, on y apprend le développement en divers langages de programation tel que le Java
  • Le SISR est une spécialité d'entretien de serveurs réseaux
